# response-collapse

Reproduction code for **"Where Corrigibility Lives in a Converged Predictor:
Interventional Leverage, Priced Plasticity, and an Adaptivity Audit."**

Every scalar reported in the paper is regenerated by a single fixed-seed driver.
No external data. Pure NumPy.

## Run

```bash
pip install -r requirements.txt
python reproduce.py --quick    # qualitative reproduction, ~2 minutes
python reproduce.py --full     # every reported scalar (~15-25 min, single core)
python reproduce.py --check    # compare a --full run against reference values
python verify_package.py       # fail loudly if any promised file is missing
python make_figures.py         # regenerates both paper figures from results
```

Runtime: a few minutes on a laptop, single core.

## Files
- `skeleton.py`   — Dirichlet-histogram posterior for a slowly-learnable source
- `leverage.py`   — finite-perturbation interventional leverage L_X
- `experiments.py`— experiment blocks E1–E7
- `robustness.py`  — E9 (α-sweep, σ-plateau, priors, truth families, bootstrap CIs)
- `learned_memory.py` — E8/E10 non-Bayesian learned-system audit + planted audit
- `verify_package.py`, `tests/`, `results_reference/` — self-verification
- `reproduce.py`  — one-command driver (`--quick` / `--full`); writes `results/`
- `make_figures.py` — regenerates the two paper figures from results
